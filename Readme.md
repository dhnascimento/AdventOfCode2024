# Advent of Code 2024

## Setup
1. Clone the repository
2. Create input files in the `inputs/` directory (e.g., `day01.txt`, `day02.txt`)
3. Implement solutions in `src/AdventOfCode2024/Solutions/`

## Running Solutions
```bash
dotnet run --project src/AdventOfCode2024 Day1
```

## Running Tests
```bash
dotnet test
